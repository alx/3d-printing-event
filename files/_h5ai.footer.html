sketchup_dxf_to_stl_plugin.rb

Plugin à rajouter dans Sketchup afin de pouvoir exporter les objets 3d en STL pour leur utilisation dans skeinforge.

A placer dans le répertoire:

  • sur Mac: /Library/Application Support/Google Sketchup 8/SketchUp/plugins/
  • sur Linux avec Sketchup dans Wine: ~/.wine/drive_c/Program Files/Google/Google Sketchup 8/Plugins/
