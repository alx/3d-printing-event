Pourquoi? Comment?

Durant la Novela 2011, des personnes des Pays-Bas de Ultimaker et Protoscape nous ont donné leur recette pour faire partager leurs expériences avec des imprimantes 3D: se retrouver une fois toutes les 6 semaines

Cet évènement est gratuit et ouvert à tous.

Participants

  • Vous avez une imprimante 3D et vous voulez partager vos connaissances
  • Vous voulez vous renseigner sur les technologies d'impression 3D
  • Vous voulez monter votre imprimante 3D
  • Association Planète Sciences Midi-Pyrénées
  • Association Tetalab - Hackerspaces.org
  • Association Artilect - Fablab

Historique

Tetalab

samedi 3 décembre 2011

Voici une description de ce qui s'est passé pendant la première session des 3d-printing-event.

  • le matin, une dizaine de personnes, 2 ultimakers et 1 makerbot, bonne ambiance, pas mal de gens nouveaux qui se sont rencontrés et ont découvert un nouveau lieu.
  • l'après-midi, en mode atelier:
    • Alex: installation de Marlin_v1, configuration de skeinforge, utilisation de PrintRun et impression de quelques pièces de test
    • Arne: robot Deuligne, avec programmation des steps du robot directement sur le Deuligne
    • Lio: accelerometre sur jante de velo pour DIY revolight

Impressions

  • Facade pour Objective2 : impression réussie
  • Rotary engine : problème d'impression avec décalage sur axe X à 5mm de hauteur.
  • BIC upgrade : impression non terminée, objet pour tester les problème de l'impression de petits détails

Profiles Skeinforges

Pour skeinforge, j'ai créé un repo git afin de pouvoir facilement récupérer les paramètres qui fonctionnent pour un type d'impression et un type de machine:

https://github.com/alx/skeinforge-dotfile

L'utilisation des fonctionnalités de git pourrait permettre de créer différentes config d'impressions rapidement utilisables sur différentes machines.
